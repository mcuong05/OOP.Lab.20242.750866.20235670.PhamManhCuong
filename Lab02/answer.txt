If you create a constructor method to build a DVD by title then create a constructor method to build a DVD by category. Does JAVA allow you to do this ?
Trong Java, ta có thể định nghĩa nhiều constructor trong cùng một class thông qua constructor overloading – tức là nạp chồng hàm khởi tạo miễn là chúng có danh sách tham số khác nhau (về số lượng hoặc kiểu dữ liệu). Tuy nhiên, nếu 2 constructor cùng dùng 1 tham số String, thì Java sẽ báo lỗi vì không phân biệt được cái nào dùng cho title, cái nào dùng cho category. 
=> Nên nếu ta muốn có 2 constructor cùng dùng tham số String thì ta phải thay đổi tên biến hoặc thay đổi cấu trúc tham số
